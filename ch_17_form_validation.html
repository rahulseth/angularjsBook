Chapter 17 AngularJS form validation

  • AngularJS can validate input data.
  • They give us various information about a form or its inputs and are applied to a form and inputs.
  • various properties are:-
    • PROPERTY : $valid CLASS : ng-valid Boolean Tells whether an item is currently valid based on the rules you placed.
    • PROPERTY : $invalid CLASS : ng-invalid Boolean Tells whether an item is currently valid based on the rules you placed.
    • PROPERTY : $pristine CLASS : ng-pristine Boolean True if the form/input has not been used yet.
    • PROPERTY : $dirty CLASS : ng-dirty Boolean True if the form/input has been used.
    • PROPERTY : $touched CLASS : ng-touched Boolean True if the input has been blurred.
