Chapter 11 AngularJS Filter

Filters can be added in AngularJS to format data.
    AngularJS provides filters to transform data:
  • currency Format a number to a currency format.
  • date Format a date to specified string.
  • filter Format a number to a currency format.
  • orderBY Format a number to a currency format.
  • uppercase Format a string to uppercase.
  • lowercase Format a string to lowercase.
  • limitTo Limit array/string to specified number of element/characters.
  • json Format a object to a json string.
Filters can be added to expressions by using the pipe character |, followed by a filter.

currency {{ price | currency }}
date {{ today | date }}
upercase {{ name | uppercase }}
lowercase {{ name | lowercase }}
json 
 {{skills | json : 2}}
