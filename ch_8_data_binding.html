Chapter 8 Data Binding

  • Data binding in AngularJS is synchronization between model & view.

  • Two way data binding
    when data in model changes, view reflect the change, & when data in view change, model also update as well. This happen automatically & immediately which make sure model & view updated at all time.


  • One way data binding
    Model value automatically assigned to HTML element specified through data binding. But HTML element can't change the value of Model.
